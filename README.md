# mondayDB Agentic Evolution

This repository captures product and engineering proposals for evolving mondayDB
from a high-performance WorkOS engine into an enterprise-grade agentic database.

## Current specs

- [Agentic Memory Layer](docs/agentic-memory-layer.md): deterministic procedural
  and semantic memory for agents, with tenant isolation, auditability, vector
  retrieval readiness, and query guardrails.
