# mondayDB Agentic Evolution

This repository tracks product and engineering artifacts for evolving mondayDB
from a high-performance WorkOS engine into an enterprise-grade agentic database.

## Documents

- [Agentic Memory Retrieval Plane](docs/agentic-memory-retrieval-plane.md):
  tenant-scoped procedural and semantic memory design, including SQL and
  TypeScript schemas, GraphQL API shape, vector retrieval guardrails, and
  performance checks for large boards.
